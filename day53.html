1. requests + BeautifulSoup

For pages that serve complete HTML (no JavaScript rendering needed), requests fetches the page and BeautifulSoup parses it. This covers the majority of scraping targets.

$ pip install requests beautifulsoup4 lxml
Python — basic scraping 
import requests
from bs4 import BeautifulSoup

def fetch_page(url: str) -> BeautifulSoup:
    headers = {
        "User-Agent": "Mozilla/5.0 (compatible; MyBot/1.0; +https://mysite.com/bot)"
    }
    response = requests.get(url, headers=headers, timeout=10)
    response.raise_for_status()
    return BeautifulSoup(response.text, "lxml")   # lxml is faster than html.parser

soup = fetch_page("https://books.toscrape.com/")

# Find elements
title = soup.find("h1").text.strip()

# CSS selector — same syntax as JavaScript's querySelector
books = soup.select("article.product_pod")
for book in books:
    name  = book.select_one("h3 a")["title"]       # attribute access
    price = book.select_one(".price_color").text
    rating = book.select_one("p.star-rating")["class"][1]  # "One", "Two", etc.
    print(f"{name}: {price} ({rating} stars)")

Navigating the Parse Tree

Python 
from bs4 import BeautifulSoup, Tag

html = """


  
Python Cookbook

  $39.99
  
  • python
  • programming



"""
soup = BeautifulSoup(html, "lxml")

# find() returns first match; find_all() returns a list
product = soup.find("div", class_="product")

# Navigate parent/siblings/children
title_tag = product.find("h2")
print(title_tag.text)               # "Python Cookbook"
print(title_tag.next_sibling)       # whitespace text node
print(title_tag.find_next("span"))  # $39.99

# Get all list items
tags = [li.text for li in product.select("ul.tags li")]
print(tags)   # ["python", "programming"]

# Attribute access
link = soup.find("a")
if link:
    print(link.get("href", "no href"))   # safe attribute access

# Check if element exists before accessing
price_tag = product.find("span", class_="price")
price = price_tag.text if price_tag else "N/A"

2. Handling Pagination

Python — paginated scraper 
import requests
import time
from bs4 import BeautifulSoup
from dataclasses import dataclass

@dataclass
class Book:
    title: str
    price: str
    rating: str

def scrape_books(base_url: str, max_pages: int = 5) -> list[Book]:
    books = []
    page = 1

    while page <= max_pages:
        # books.toscrape.com uses catalogue/page-N.html
        url = f"{base_url}/catalogue/page-{page}.html"
        response = requests.get(url, timeout=10)

        # Stop if we've gone past the last page
        if response.status_code == 404:
            break
        response.raise_for_status()

        soup = BeautifulSoup(response.text, "lxml")
        articles = soup.select("article.product_pod")

        if not articles:
            break

        for article in articles:
            books.append(Book(
                title=article.select_one("h3 a")["title"],
                price=article.select_one(".price_color").text,
                rating=article.select_one("p.star-rating")["class"][1],
            ))

        # Find next page link — stop if there isn't one
        next_btn = soup.select_one("li.next a")
        if not next_btn:
            break

        page += 1
        time.sleep(1)   # be polite — 1 request per second
        print(f"Scraped page {page-1}, {len(books)} books so far")

    return books

books = scrape_books("https://books.toscrape.com", max_pages=3)
print(f"Total: {len(books)} books")

3. Playwright — JavaScript-Rendered Pages

Many modern sites render content with JavaScript after the initial page load — BeautifulSoup only sees the raw HTML, which has no data yet. Playwright controls a real browser, waits for JavaScript to run, then lets you extract the rendered content.

$ pip install playwright
$ playwright install chromium # download browser binaries
Python — Playwright sync API 
from playwright.sync_api import sync_playwright
from dataclasses import dataclass

@dataclass
class Listing:
    title: str
    price: str
    url: str

def scrape_with_playwright(search_url: str) -> list[Listing]:
    listings = []

    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)   # headless=False to watch it work
        page = browser.new_page()

        # Set realistic headers
        page.set_extra_http_headers({
            "User-Agent": "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)"
        })

        page.goto(search_url)

        # Wait for specific element to appear — auto-waits up to 30s
        page.wait_for_selector(".listing-card", timeout=10_000)

        # Get all listing cards
        cards = page.query_selector_all(".listing-card")
        for card in cards:
            title = card.query_selector("h2")
            price = card.query_selector(".price")
            link  = card.query_selector("a")
            if title and price and link:
                listings.append(Listing(
                    title=title.inner_text().strip(),
                    price=price.inner_text().strip(),
                    url=link.get_attribute("href") or "",
                ))

        browser.close()
    return listings

Playwright Async API (for FastAPI / asyncio)

Python 
import asyncio
from playwright.async_api import async_playwright

async def scrape_async(url: str) -> str:
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()
        await page.goto(url)
        await page.wait_for_load_state("networkidle")  # wait for all XHR to finish
        content = await page.content()   # full rendered HTML
        await browser.close()
        return content

# Useful Playwright actions
async def interact_with_page(page):
    await page.fill("#search-input", "python books")    # type into input
    await page.click("#search-button")                  # click a button
    await page.wait_for_selector(".results")            # wait for results
    await page.screenshot(path="screenshot.png")        # save screenshot for debugging
    text = await page.inner_text(".results")            # get text content

4. Rate Limiting + robots.txt

Python — polite scraper 
import time
import random
from urllib.robotparser import RobotFileParser
from urllib.parse import urlparse
import requests

def is_scraping_allowed(url: str, user_agent: str = "*") -> bool:
    """Check robots.txt before scraping."""
    parsed = urlparse(url)
    robots_url = f"{parsed.scheme}://{parsed.netloc}/robots.txt"
    rp = RobotFileParser()
    rp.set_url(robots_url)
    rp.read()
    return rp.can_fetch(user_agent, url)

class PoliteScraper:
    def __init__(self, min_delay: float = 1.0, max_delay: float = 3.0):
        self._min_delay = min_delay
        self._max_delay = max_delay
        self._session = requests.Session()
        self._session.headers["User-Agent"] = "MyBot/1.0 (+https://example.com/bot)"

    def get(self, url: str) -> requests.Response:
        if not is_scraping_allowed(url):
            raise PermissionError(f"robots.txt disallows scraping: {url}")

        # Random delay — looks more human, avoids hammering the server
        delay = random.uniform(self._min_delay, self._max_delay)
        time.sleep(delay)

        response = self._session.get(url, timeout=10)
        response.raise_for_status()
        return response

scraper = PoliteScraper(min_delay=1.5, max_delay=4.0)
# response = scraper.get("https://example.com/products")
Ethical scraping checklist:
✅ Check robots.txt and respect it
✅ Add 1–3 second delays between requests
✅ Identify your bot in the User-Agent header
✅ Respect rate limit headers (Retry-After, X-RateLimit-*)
✅ Only scrape publicly accessible data
✅ Cache responses — don't re-fetch what you already have
❌ Don't scrape personal data without consent
❌ Don't bypass authentication or paywalls

Quick Reference

Fetch page
BeautifulSoup(requests.get(url).text, "lxml")

Use lxml parser — faster than html.parser

CSS selector
soup.select("div.product h2")

Returns list; select_one() returns first or None

Get text
tag.text.strip()

Always strip — HTML often has leading/trailing whitespace

Get attribute
tag["href"] or tag.get("href", "")

tag["href"] raises KeyError if missing; .get() is safe

Playwright launch
p.chromium.launch(headless=True)

headless=False to watch the browser during development

Wait for element
page.wait_for_selector(".results", timeout=10_000)

Waits up to 10s; raises TimeoutError if not found

Polite delay
time.sleep(random.uniform(1, 3))

Random delay looks more human and avoids triggering rate limits

Check robots.txt
RobotFileParser().can_fetch(user_agent, url)

Always check before scraping — respect the site's wishes

Wrap-up

  • Use requests + BeautifulSoup for static HTML pages. Use Playwright only when JavaScript rendering is required — it's 10–100× slower than a simple HTTP request.
  • CSS selectors (soup.select("div.card h2 a")) are more concise and familiar than BeautifulSoup's find() chaining. If you know CSS, you already know how to navigate the DOM.
  • Always check robots.txt before scraping. It's both a legal and ethical boundary — many sites explicitly disallow scrapers in their terms of service as well.
  • Random delays (1–3 seconds) between requests are not just polite — they also reduce the chance of being rate-limited or IP-banned. Consistent 1-second delays are easier for anti-scraping systems to detect.
  • Playwright's wait_for_selector() auto-waits for elements to appear — unlike Selenium where you had to manage WebDriverWait and ExpectedConditions manually. Much cleaner.
